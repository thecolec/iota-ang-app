Iota

Simplifying Student Organizations
Why Iota?

The disorganization of student organizations is the result of the lack of tools.
This project was an opportunity for me to explore the standards of modern web development,
as well as develop new tools for the organizations I love.

What is Iota?

Iota is an asynchronous, stateless, Progressive Web App (PWA).
It is built with Angular for the Front-End, and a custom API Server running in Node.JS on the Back-End.
It employs modern development standards seen in environments such as Facebook, Amazon, Netflix, etc.

What next?

The goal is to grow Iota over the next few years after graduation.
I firmly believe that continued development will make this an essential tool for all students/faculty.
Not just those involved in greek life.
